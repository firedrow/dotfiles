# dotfiles

Here are copies of my linux dotfiles, backed up to Github and installed using [GNU stow](https://taihen.org/managing-dotfiles-with-gnu-stow/).

## Intel Stick - Ubuntu

stow git
stow nano
stow ssh
stow zsh

## Virtualbox - Ubuntu

stow git
stow nano
stow openbox
stow ssh
stow terminator
stow tint2
stow zsh
