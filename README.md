# dotfiles

Here are copies of my linux dotfiles, backed up to Github and installed using [GNU stow](https://taihen.org/managing-dotfiles-with-gnu-stow/).

## Intel Stick - Ubuntu

stow git <br />
stow nano <br />
stow ssh <br />
stow zsh

## Virtualbox - Ubuntu

stow git <br />
stow nano <br />
stow openbox <br />
stow ssh <br />
stow terminator <br />
stow tint2 <br />
stow zsh
